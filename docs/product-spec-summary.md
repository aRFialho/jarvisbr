# Resumo da especificacao DOCX

O arquivo original esta versionado em [product-spec.docx](product-spec.docx).

Este repositorio implementa o MVP 0 e o MVP 1:

- API Render + Neon.
- SQL puro sem Prisma.
- Auth, aparelhos, pareamento, WebSocket, comandos e confirmacoes.
- Agent Windows com indice local SQLite e busca fuzzy.
- Web/painel holografico funcional.
- Base mobile Flutter.
- Worker implementado, mas nao ativado por padrao para evitar custo no Render.
